## func ls

  Usage:	hyper func ls [OPTIONS]

  Lists all funcs

    -f, --filter=[]    Filter output based on conditions provided
    --help             Print usage

### Examples

    $ hyper service ls
    NAME         SIZE     IMAGE           COMMAND          CREATED           UUID
